Методы

add

Описание метода

Добавляет новый тикет.


Поля

Имя поля Необходимость Тип данных Комментарий
text True string Текст обращения в техническую поддержку.
Максимум 2000 символов.
subject False string Тема обращения в техническую поддержку.

Результат

Ticket

Доступы к методу

Имя роли доступ Комментарий
admin full
manager full
chief full
chief_partner full
operator full
admin_partner full

list

Описание метода

Возвращает список тикетов.

Поля

Имя поля Необходимость Тип данных Комментарий
q False string Критерий поиска.
Доступные поля:
ids – idlist, список ID тикетов;
employee_ids – idlist, список ID сотрудников;
type;
priority;
status;
created_at.
fields False string Список через запятую возвращаемых полей.
limit False numeric По умолчанию – 50.
sort False string Сортировка результатов.
Возможные значение:
created_at:d – по умолчанию;
type:a, type:d;
priority:a, priority:d;
status:a, status:d.
offset False numeric По умолчанию – 0.

Результат

Array.<Ticket>

Доступы к методу

Имя роли доступ Комментарий
admin full
manager full
chief full
chief_partner full
operator user Только свои тикеты
admin_partner full

id True numeric ID обращения в техническую поддержку.
