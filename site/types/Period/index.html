Period

Описание типа

Период.

Контроллер: Periods.

Поля

Имя поля Необходимость Тип данных Комментарий
vat_invoices False Array.<VATInvoice> Массив счетов-фактур.
carts False Array.<Cart> Массив корзин.
created_at True datetime Дата создания.
is_autorenewal True boolean Признак автоматического продления.
updated_at True datetime Дата последнего обновления.
id True numeric ID периода.
invoices False Array.<Invoice> Массив счетов.
products True Array.<Product> Массив продуктов.
period_state True string Статус периода.
Возможные значения:
expired – прошедший;
active – активный;
next – будущий.
acceptances False Array.<Acceptance> Массив актов.
orders False Array.<Order> Массив заказов.
