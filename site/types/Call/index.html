Call

Описание типа

Звонок.

Контроллер: Сalls.

Поля

Имя поля Необходимость Тип данных Комментарий
employee_vote False EmployeeVote Оценка сотрудника.
visitor True Visitor Посетитель.
page_count False numeric Количество страниц, просмотренных посетителем в ходе визита, перед тем, как он совершил звонок.
rem_lost_in_queue True boolean Признак "Потерянный в очереди".
В данный момент всегда false.
rem_transfer True boolean Признак "Ручной перевод".
result True string Итог звонка.
Возможные значения:
missed – пропущенный;
completed – состоявшийся.
duration True numeric Продолжительность звонка в секундах, включая answer_time.
visit_count False numeric Общее количество визитов посетителя на сайт, включая визит, в котором был совершен звонок.
id True numeric ID звонка.
prechats_chat False Array.<Object> Пречат поля, видимые в Пульте оператора.
Эти поля могут быть настроены в Личном кабинете. В этом случае их значение указывает посетитель при старте чата.
Кроме того, пречат поля могут быть программно определены в параметрах chatAttributes методов Client API: .
Значение поля – массив объектов, каждый из которых имеет 2 ключа: name и value, содержащие соответственно имя и значение пречат поля.
answer_time True numeric Время в секундах, через которое оператор ответил на ответил на звонок.
lead False Lead Лид, из которого был сделан звонок.
Имеет смысл только для исходящих звонков.
rem_long_answer True boolean Признак "Долгий ответ".
В данный момент всегда false.
visitor_vote False string Оценка посетителя.
Возможные значения:
positive – положительная оценка;
negative – отрицательная оценка;
undefined – посетитель не поставил оценку.
employee True Employee Сотрудник.
ext_referer False string Адрес, с которого посетитель перешел на сайт.
prechats_hidden False Array.<Object> Скрытые пречат поля. В Пульте оператора они не отображаются.
Пречат поля этого типа могут быть программно определены в параметрах hiddenAttributes методов Client API: .
Значение поля – массив объектов, каждый из которых имеет 2 ключа: name и value, содержащие соответственно имя и значение пречат поля.
search_engine False string Название поискового сервиса.
Содержит значение, если посетитель начал звонок, перейдя на сайт из результатов поиска сервисов типа Yandex или Google.
is_managed True boolean True, если звонок входит в число своих звонков сотрудника, вызывающего метод.
Это признак доступен только для чтения.
updated_at True datetime Дата последнего обновления.
department False Department Отдел.
rem_empty True boolean Признак "Отсеянный".
rem_auto True boolean Признак "Автоматический перевод".
int_referer False string Адрес страницы, на которой начался диалог.
is_incoming True boolean Признак входящего звонка.
search_query False string Строка поискового запроса.
Содержит значение, если посетитель начал звонок, перейдя на сайт из результатов поиска сервисов типа Yandex или Google.
created_at True datetime Дата создания.
recording False string URL записи разговора.
conversation False Conversation Обращение, в рамках которого состоялся звонок.
site True Site Сайт.
callback_url False string Значение настройки Callback URL на момент начала чата.
entry_page False string Страница, с которой посетитель начал визит, в котором было отправлен лид.
