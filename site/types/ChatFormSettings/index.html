ChatFormSettings

WARNING: DEPRECATED

Описание типа

ВНИМАНИЕ! Сущность является устаревшей. Все переехало в WidgetSettings с существенными изменениями.
Настройки внешнего окна чата.
Эти настройки управляют внешним видом ярлыка, встроенного и внешнего окна.

ПРИМЕЧАНИЯ:
Сервис LiveTex предполагает возможность индивидуального дизайна окна чата для конкретного сайта. В этом случае значение настроек могут не соответствовать действительности и их нельзя изменить ни в личном кабинете, ни с помощью методов Web API.

Поля

Имя поля Необходимость Тип данных Комментарий
size True string Размер ярлыка.
Возможные значения:
small – маленький, 24x106 пикселей;
large – большой, 40x178 пикселей (по умолчанию).
offset_value True numeric Значение отступа.
Целое число в диапазоне:
от 0 до 100 для offset_type = percent;
от 0 до 2000 для offset_type = pixel.
color False string Цветовая схема.
Возможные значения:
green – зеленая;
orange – оранжевая;
blue – синяя;
red – красная;
purple – фиолетовая;
gray – серая;
rose – розовая;
black – черная;
yellow – желтая;
white – белая.
Поле обязательно для color_type = preset.
background_typе True string Тип фона.
Возможные значения:
none – без фона;
default – стандартный фон;
custom – загружаемый фон.
Актуально только для внешнего окна.
color_type True string Тип указания цветовой схемы.
Возможные значения:
preset – стандартная схема;
custom – явное указание цветов.
updated_at True datetime Дата последнего обновления.
is_hidden_offline True boolean Скрывать ярлык, если сотрудники офлайн.
color_main False color Основной цвет.
Поле обязательно для color_type = custom.
banner_type True string Тип баннера.
Возможные значения:
none – без баннера;
default – стандартный баннер;
custom – загружаемый баннер.
Актуально только для внешнего окна.
is_custom True boolean Признак заказного дизайна ярлыка и окна чата.
Если true, то изменение некоторых настроек может не иметь должного результата, поскольку соответствующий аспект внешнего вида переопределяются заказным дизайном.
Это признак доступен только для чтения.
color_background False color Цвет фона.
Поле обязательно для color_type = custom.
banner_custom False file Загруженная баннер для banner_type = custom.
Изображение в формате JPEG, GIF или PNG с шириной от 1 до 448 px и высотой 82 px.
Поле обязательно для banner_type = custom.
Актуально только для внешнего окна.
banner_link False string Ссылка с баннера.
Поле обязательно для banner_type = custom.
Актуально только для внешнего окна.
position True string Положение ярлыка.
Возможные значения:
top – сверху;
right – справа;
bottom – снизу;
left – слева.
is_label_visible True boolean Признак отображения ярлыка.
По умолчанию – true.
Если false, ярлык не будет показываться посетителю, даже при наличии доступных операторов.
Это признак автоматически устанавливается в true и не может быть изменен, если подключена опция генератора лидов или для сайта включено встроенное окно чата (site.is_embedded_chat == true).
Признак is_label_visible связан с полем "size". При переключении из false в true поле "size" устанавливается в значение по умолчанию.
logo_type True string Тип логотипа LiveTex.
Возможные значения:
animated – анимированный;
static – статический.
offset_type True string Тип отступа.
Возможные значения:
pixel – в пикселах;
percent – в процентах.
background_custom False file Загруженный фон для background_type = custom.
Изображение в формате JPEG, GIF или PNG с размерами 448x605px.
Поле обязательно для background_type = custom.
Актуально только для внешнего окна.
photo_size True string Размер фотографии сотрудника.
Возможные значения:
small – маленькая, 60x70 пикселей;
large – большая, 100x116 пикселей.
color_text False color Цвет текста.
Поле обязательно для color_type = custom.
