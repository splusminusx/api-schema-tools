## addToPeriod

### Описание метода
Carts.addToPeriod
Создает новую корзину в указанном периоде.
Параметры
Результат
Объект типа «Cart».
Уровень доступа для ролей


### Поля

| Имя поля | Необходимость | Тип данных | Комментарий |
|---|---|---|---|
|activation_type|False|None|Тип активации.<br/>Возможные значения:<br/>now – активация сразу после покупки (по умолчанию);<br/>after_current – активация после завершения текущего периода;<br/>after_payment – активация после оплаты.<br/>|
|period_id|True|None|ID периода.<br/>|

## commit

### Описание метода
Carts.commit
Операция покупки позиций указанной корзины.
Метод создает заказ, выставляет счет на оплату на активного плательщика и удаляет корзину. 
 Параметры
Результат
Объект типа «Order».
Уровень доступа для ролей


### Поля

| Имя поля | Необходимость | Тип данных | Комментарий |
|---|---|---|---|
|id|True|None|ID корзины.<br/>|
|payer_id|False|None|ID плательщика.<br/>Если не указано, то счет будет выписан на активного плательщика.<br/>|

### Резудьтат
None
### Доступы к методу

| Имя роли | доступ | Комментарий |
|---|---|---|
|admin|manager|chief|chief_partner|operator|admin_partner
